How to use:
1. Click Upload disassembly to load a .txt / .asm / .dis file.
2. Supports two disassembly formats:
  • asm ; address | opcode
  • address opcode asm (address may include 'H' suffix)
3. Enter instructions to search (one per line).
4. Wildcard $ matches any decimal number (0-99) in register names.
5. Without $, matching is prefix-based.
6. Click Find. Each result ends at pop pc, rt, or rti.
7. Use Sort by address / length to reorder. Copy all copies every found gadget.
